# VibeCheck+

**VibeCheck+** is a technically substantial Chrome Extension (Manifest V3) designed to combat emotional overload and toxic content exposure on social media. Unlike simple sentiment blockers, VibeCheck+ uses **Adaptive Emotional Filtering** and **Focus Mode** to create a personalized digital sanctuary.

## 🚀 Unique Innovation: Adaptive Focus Mode

Social media fatigue isn't just about one bad post; it's about the *density* of negativity. VibeCheck+ tracks your "Emotional Load" in real-time. If it detects you are being bombarded by aggressive or toxic content, it automatically suggests or activates **Focus Mode**, which temporarily increases filtering strictness to help you decompress.

## ✨ Core Features

1.  **Real-time DOM Analysis**: Modular scanning for X/Twitter, Reddit, and LinkedIn using `MutationObserver`.
2.  **Local-First NLP Engine**: Hybrid rule-based and scoring system that runs entirely on your device for maximum privacy.
3.  **Multidimensional Emotional Scoring**:
    - Anger, Sadness, Toxicity, Aggression, and Informational scores.
4.  **Adaptive Filtering**: Content is blurred based on dynamic thresholds. A "Reveal" button allows for conscious consumption.
5.  **Emotional Exposure Dashboard**:
    - **Total Scanned**: Cumulative count of posts processed.
    - **Emotion Distribution**: Pie chart showing the emotional "vibe" of your feed.
    - **Daily Trend**: Line graph tracking your exposure to negativity over time.
    - **Load Score**: A calculated metric of current emotional stress based on recent browsing.
6.  **Whitelist System**: Personalize your experience by whitelisting specific users or keywords.

## 🏗️ Architecture

-   **DOM Scanner (`content.ts`)**: Batched processing of feed items to ensure 0% performance impact on scrolling.
-   **NLP Engine (`localEngine.ts`)**: Analyzes text and returns a granular emotional profile.
-   **Background Service (`background.ts`)**: Manages session state, global analytics, and Focus Mode timers.
-   **Storage (`storage.ts`)**: Uses `chrome.storage` for persistence, ensuring your data never leaves your machine.
-   **UI Dashboard (`popup.html/ts`)**: Built with Chart.js for beautiful, actionable analytics.

## 🛠️ Build & Installation Guide

### Prerequisites
-   Node.js (v18+)
-   npm

## Download
[Releases Download](https://github.com/kimrasng/VibeCheck/releases)

### Loading into Chrome
1.  Open Chrome and go to `chrome://extensions/`.
2.  Enable **Developer mode** (top right).
3.  Click **Load unpacked**.
4.  Select the `dist` folder generated by the build process.

## 📸 Screenshot Guide
To capture screenshots for submission:
